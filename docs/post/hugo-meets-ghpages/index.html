hugo meets ghpages

30 April 2017

Overview

hugo와 github page로 블로그를 만들어보자

1. hugo
    - static site generator. markdown 문서를 html로 바꿔준다고 생각하자
    - theme을 생성하거나 적용할 수 있으며, local 에서 블로그를 동작시켜 볼 수 있다

2. github page
    - github에 repository를 만들어 github page, 즉 블로그로 이용할 수 있다.

3. hugo meets github
    - hugo로 생성한 블로그를 git을 사용하여 github page에 올리고 관리한다
